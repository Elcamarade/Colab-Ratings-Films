# 🎬 Colab Ratings Films

## 📌 Descrierea proiectului

**Colab Ratings Films** este o aplicație web care permite utilizatorilor să caute filme și să vadă rating-ul și o scurtă descriere despre acestea.
Aplicația folosește un **API extern** pentru a prelua informațiile despre filme și pentru a le afișa pe site.

Scopul proiectului este de a demonstra utilizarea:

* HTML
* CSS
* JavaScript
* API-uri externe

---

## 🚀 Cum se deschide proiectul

1. Descarcă sau clonează repository-ul de pe GitHub:

```
git clone https://github.com/nume-user/Colab-Ratings-Films.git
```

2. Intră în folderul proiectului:

```
cd Colab-Ratings-Films
```

3. Deschide fișierul **index.html** într-un browser
   sau deschide proiectul în **Visual Studio Code**.

4. Introdu numele unui film în bara de căutare pentru a vedea informațiile despre acesta.
